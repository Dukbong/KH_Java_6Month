### Eclipse 사소한 팁
---


- **글자 크기 확대 및 축소**
	- ctrl + "+" (확대)
	- ctrl + "-" (축소)
  
  
- **변수명 한번에 바꾸기**
	- 방법 1) alt + shift + r
	- 방법 2) 우클릭 -> Refactor -> rename


- **자동 import**
	- ctrl + shift + o


- **소스 코드 이동**
	- 드래그 + alt + 방향키


- **소스 코드 빠른복사**
	- 드래그 + ctrl + alt + 방향키
